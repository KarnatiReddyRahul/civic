# Security Policy

## Supported Versions

The following versions of this project are currently supported with security updates:

| Version | Supported          |
| ------- | ------------------ |
| 1.0.x   | :white_check_mark: |

## Reporting a Vulnerability

We take the security of CivicAssist AI seriously. If you find a security vulnerability, please do not report it in the public issue tracker.

Instead, please send an email to the project maintainers or report it privately through GitLab's vulnerability reporting feature. We will investigate the issue and provide a fix as soon as possible.
